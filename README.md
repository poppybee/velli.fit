# VELLI Sportswear (velli.fit)

A modern, high-performance women's athletic wear e-commerce site built with Next.js 16, Tailwind CSS 4, and Framer Motion.

## Features
- Dynamic Hero section with motion effects.
- Categorized product browsing.
- Detailed product pages with size selection and image galleries.
- Full checkout flow (Cart, Shipping, Payment, Success).
- Integrated user account and login UI.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Language**: English (International Market ready)

## Getting Started
Run the development server:
```bash
npm run dev
```
Or build and start for production:
```bash
npm run build
npm start
```
